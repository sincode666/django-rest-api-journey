# Step 01: What is an API?

## Overview
An API (Application Programming Interface) enables communication between different software systems. In web development, it connects the frontend (client) with the backend (server).

## Core Concept
The API follows a request-response cycle:
- Client sends a request
- Server processes it
- Server returns a response

## Example Flow
Request:
Client → "Give me student data"
Response:
Server → Returns JSON data
## Key Idea
APIs act as a bridge between systems without exposing internal logic.
## Real-World Example
When you open a weather app, it calls an API to fetch live weather data from a server.
