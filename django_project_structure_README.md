# Step 03: Django Project Structure

## Overview
Django separates functionality into projects and apps.

## Components

### Project
- Main configuration
- Contains settings and root URLs

### App
- Handles specific functionality
- Example: students app

## Important Files

- models.py → database structure
- views.py → business logic
- urls.py → routing

## Flow

Request → URL → View → Response

## Key Idea
Clear separation of concerns improves maintainability.
