# Step 02: HTTP Methods

## Overview
HTTP methods define the type of operation performed on data in an API.

## Methods

### GET
- Used to retrieve data
- Does not modify data

### POST
- Used to create new data
- Sends data in request body

### PUT
- Used to update existing data
- Replaces full resource

### DELETE
- Used to remove data

## Example

GET /students → fetch all students  
POST /students → create student  
PUT /students/1 → update student  
DELETE /students/1 → delete student  

## Key Idea
These methods form the foundation of REST APIs.
